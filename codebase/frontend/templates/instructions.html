About CensusConnect

CensusConnect currently works with ACS Profile tables. Data Profiles are an excellent starting point for novice data users, containing the most frequently requested social, economic, housing, and demographic data.

Data Profiles summarize data for a single geographic area, providing both estimates and percentages, covering the most basic data on all ACS topics. ACS Data Profiles begin with the letters "DP."

Learn more about Census Tables

Requesting Data

On the Request Data page, you can consult the AI assistant at any time for help. The process involves:

  • Selecting variables from dropdown boxes
  • Choosing desired years
  • Selecting survey type
  • Specifying geographic area

Managing Projects

Users can create new projects or work with existing ones. Within projects, you can:

  • View saved searches specific to the project
  • Write and maintain project notes
  • Communicate with an AI assistant that tracks project details
  • Get help with potential data issues

All saved searches can be accessed through the Saved Searches tab for easy reference.
